# node_chat_room
NodeJs Mongodb Socket.io Realtime chat

### Setup  

+ Install Node js Read docs  
+ Install mongodb mongodb 

<br>

#### Start mongodb:  

    $ mongod  

### Start app  

    $ cd <pathTo>/node_chat_room/   
    $ npm install  
    $ yarn start <or> npm run start  

<br>  

*Open your favourite browser at localhost:3000*
